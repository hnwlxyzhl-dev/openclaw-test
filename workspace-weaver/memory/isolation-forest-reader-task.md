# 读者评审子代理任务

你是一个有基础数据分析背景、但不熟悉异常检测算法的研究生。请评审这份 PPT，假设你是目标读者。

## 必读文件
1. /home/admin/.openclaw/workspace-weaver/output/isolation-forest_v6.pptx（用 python-pptx 读取所有文本内容）
2. /home/admin/.openclaw/workspace-weaver/memory/isolation-forest-ppt-content-logic.md（原始内容设计）

## 评审要求
逐页阅读 PPT 的文本内容，站在读者角度评审：
1. 这一页我能看懂吗？
2. 和上一页的衔接自然吗？
3. 比喻是否帮助理解？还是反而让人困惑？
4. 内容是否太浅？缺少什么关键信息？
5. 有没有文字重叠、排版问题？
6. 视觉上是否美观？有没有觉得"丑"的地方？
7. 信息密度是否合适？是太密还是太松？

## 评审维度（满分100分）
1. 内容深度（20分）— 是否过于简单？是否有足够的细节？
2. 逻辑连贯（20分）— 页面之间是否有自然的过渡？叙事是否流畅？
3. 通俗易懂（20分）— 目标读者能否看懂？比喻是否贴切？
4. 信息密度（15分）— 每页是否有足够的信息量？是否有空洞页？
5. 整体收获（15分）— 读完 PPT 后是否有实质性收获？
6. 文字质量（10分）— 是否有重叠、错别字、不通顺的句子？

## 特别关注
- 算法逻辑是否正确（公式、数值计算、伪代码）
- 比喻是否贴切，不会误导读者
- 示例数据是否前后一致
- 异常分数的计算过程是否清晰准确

## 输出格式
把评审结果写入 /home/admin/.openclaw/workspace-weaver/memory/isolation-forest-reader-report.md

包含：
- 总分（满分100）
- 每页的评审意见
- 最需要改进的5个问题
- 具体的修改建议
- 视觉设计方面的改进建议（用户觉得当前 PPT "太丑"）
