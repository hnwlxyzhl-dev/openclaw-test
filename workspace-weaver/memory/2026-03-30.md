## 00:35 - 学习powerpoint-pptx skill

**任务：** 用户要求学习powerpoint-pptx skill的使用方法

**完成：** ✅ 完成5分钟快速学习

**学习来源：**
1. 本地SKILL.md
2. Smithery官方pptx skill文档
3. python-pptx官方文档

**核心原则（6大铁律）：**

### 1. Choose the workflow before touching the deck
```
Reading/analyzing → markitdown
Editing from template → editing.md
Creating from scratch → pptxgenjs.md
```

### 2. Inventory the deck before replacing content
- 检查占位符、备注、评论、媒体
- 占位符索引不是固定的

### 3. Match content to actual placeholders
- 内容数量匹配占位符数量
- 不要强行塞入

### 4. Preserve visual language
- 保持原设计的主题、字体、对齐
- 不要发明新风格

### 5. Run content QA and visual QA separately
- Content QA：文本提取 + 检查占位符残留
- Visual QA：**必须用subagents**（即使2-3页）
- 至少一次fix-and-verify循环

### 6. Keep decks portable
- 复杂效果会跨平台降级
- 保持兼容性

**正确的PPT修复流程：**
```
1. Choose workflow (editing)
2. Inventory the deck
3. Read/analyze content
4. Edit based on inventory
5. Content QA
6. Visual QA (用subagent)
7. Fix-and-verify loop
8. Send to user
```

**我之前的错误：**
- ❌ 没有选择workflow
- ❌ 没有inventory
- ❌ 没有做proper QA
- ❌ 没有保持visual language
- ❌ 生成完就发送
