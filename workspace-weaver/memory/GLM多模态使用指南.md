### 场景4：图片优化建议

**问题：** 图片用作PPT背景时文字看不清

**解决：**
```
你："这张图片想做PPT背景，但文字看不清，请给出优化建议"
    [上传图片]

GLM建议：
1. 添加半透明蒙版
   - 黑色，透明度40-50%
   - 覆盖整个图片
   
2. 调整图片亮度
   - 降低30%亮度
   - 提高对比度20%
   
3. 文字颜色选择
   - 使用白色或浅色文字
   - 添加文字阴影
   
4. 替代方案
   - 将图片放在右侧1/2区域
   - 左侧使用纯色背景
   - 文字放在左侧

需要我提供具体操作步骤吗？
```

### 场景8：图片问题诊断

**问题：** 这张图片有什么问题？

**解决：**
```
你："这张图片用在PPT中感觉不对劲，请帮我诊断问题"
    [上传图片]

GLM诊断：
发现的问题：
1. ❌ 分辨率不足
   - 当前：800x600
   - 建议：至少1920x1080
   
2. ❌ 水印明显
   - 右下角有版权水印
   - 需要去除或更换图片
   
3. ⚠ 构图偏移
   - 主体偏向左侧
   - 影响视觉平衡
   
4. ⚠ 色彩过于鲜艳
   - 饱和度过高
   - 与商务风格不符

解决方案：
- 寻找高清无水印版本
- 或使用AI工具重新生成类似图片
- 调整色彩饱和度-20%
```
