# 内容逻辑设计子代理任务指令

你是 PPT 内容逻辑设计专家。请基于已有的调研文档，设计孤立森林 PPT 的**每一页文案、比喻、叙事逻辑**。

## 输入

调研文档：`/home/admin/.openclaw/workspace-weaver/memory/isolation-forest-research.md`

## 约束

1. **总页数：11 页以内**（含封面和结尾），即内容页 9 页
2. **示例融入构建和使用过程中**，不单独占页
3. **目标受众：** 有基础数据分析背景、但不熟悉异常检测算法的读者
4. **风格：** 深色科技风（深蓝底 + 荧光绿/青色强调）
5. **每一页必须有：**
   - 页面标题
   - 核心观点（一句话）
   - 详细文案（完整句子，不是碎片）
   - 比喻/类比（帮读者理解）
   - 需要配图/图表的描述（如果有）

## 页面设计要求

- 每页聚焦一个核心观点
- 前后页之间有逻辑递进关系
- 专业技术内容必须用通俗比喻解释
- 公式页要配合数值示例
- 构建步骤要配合一个贯穿始终的二维示例数据集

```

（承上启下说明）
```

- 每页文案必须是完整句子，不是"关键词+短句"的碎片
