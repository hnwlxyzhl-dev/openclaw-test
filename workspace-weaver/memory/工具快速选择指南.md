## 📦 快速安装包

### Windows用户

```powershell
# 1. 安装Python 3.10+（python.org下载）

# 2. 一键安装所有工具
pip install edge-tts pillow scipy

# 3. 测试语音生成
edge-tts --text "测试成功" --write-media test.mp3
```

### Mac用户

```bash
# 1. 安装Python
brew install python

# 2. 一键安装所有工具
pip3 install edge-tts pillow scipy

# 3. 测试语音生成
edge-tts --text "测试成功" --write-media test.mp3
```

## 🚀 5分钟开始制作

### 步骤1：获取图片（2分钟）
```
1. 访问 unsplash.com
2. 搜索 "business meeting"
```
