- **任务ID**: f2d586a2-6864-4030-9d75-3fbdda17429f
- **执行时间**: 每日 21:55
- **超时**: 600s
- **重要变更（最终方案）**: 使用 Python 爬虫直接获取历史数据并计算分位
  - **期货数据**: 新浪财经 API `https://stock.finance.sina.com.cn/futures/api/jsonp.php`
    - 品种: AU0(金), CU0(铜), I0(铁矿石), SC0(原油), RB0(螺纹钢), AL0(铝), ZN0(锌), NI0(镍), AG0(白银)
    - 返回格式: JSONP，需用正则 `\(\[.*\]\)` 提取 JSON 数组
    - 数据字段: d(日期), c(收盘价), o/h/l/v(开高低量)
  - **股市指数**: 东方财富 API `https://push2his.eastmoney.com/api/qt/stock/kline/get`
    - 品种: 1.000300(沪深300), 1.000001(上证), 0.399006(创业板), 100.HSI(恒生), 100.N225(日经), 100.NDX(纳指), 100.DJIA(道指), 100.GDAXI(德国DAX)
    - 参数: klt=101(日K), fqt=1(前复权), beg/end=日期范围
  - **分位计算**: 取近3年数据(约730个交易日)，计算当前价在历史中的百分位
  - **状态判断**: <25%低位🟢, 25-75%中位🟡, >75%高位🔴
