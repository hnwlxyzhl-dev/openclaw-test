### 关键认知纠正
- 之前一直以为 web_search (Brave) 能用，实测才发现根本没配 API Key
- Multi Search Engine 和 SearXNG 本质相同：都是通过服务器 IP 代理访问搜索引擎，容易触发反爬
- Brave/Tavily 是 API Key 认证，不会触发反爬——这才是稳定搜索的正确方式

### 宏观数据获取程序理解
- `MacroEconomics_push.py` 程序内部已有分行业数据获取逻辑（爬 NBS 官网 + 东财 API）
- cron 任务的 `macro_supplement_data.json` 是**额外补充数据源**，提高可靠性，不是重复造轮子
- 老板明确纠正：程序爬官网不稳定，cron + AI 搜索是重要的兜底手段
