# PPT多媒体与数据处理快速参考卡

## 📊 表格处理速查

### 快速创建
| 操作 | 方法 |
|------|------|
| 插入表格 | 插入 → 表格 → 选择行列 |
| Excel嵌入 | 复制Excel → 粘贴 → 保留源格式 |
| 精确控制 | 插入 → 表格 → 插入表格 → 输入数字 |

### 美化配色
```
商务风：深蓝表头 + 浅蓝/白交替行
科技风：深灰表头 + 渐变灰数据行
简约风：品牌色表头 + 白色数据
```

### 常用操作
```
合并单元格：布局 → 合并
调整大小：拖动边框
添加公式：布局 → 公式
条件格式：手动添加图标/色块
```

## 📁 文件格式速查

### 音频格式
```
推荐：MP3（压缩好）
高质量：WAV（无损）
PPT支持：MP3、WAV、WMA、M4A
```

### 视频格式
```
推荐：MP4（H.264+AAC）
兼容性最好
PPT支持：MP4、AVI、WMV、MOV
```

### 图片格式
```
透明背景：PNG
照片：JPG
图标：SVG（可缩放）
```

## ⚡ 快捷键速查

### 表格
```
Tab：下一个单元格
Shift+Tab：上一个单元格
Ctrl+A：选择整个表格
```

### 音视频
```
空格：播放/暂停
Alt+P：播放
Alt+End：跳到最后
```

### 通用
```
Ctrl+C/V：复制/粘贴
Ctrl+Z：撤销
Ctrl+S：保存
F5：开始放映
```

## 🎯 最佳实践检查清单

### 表格
- [ ] 字号≥12pt
- [ ] 行列对齐整齐
- [ ] 颜色≤3种
- [ ] 关键数据突出
- [ ] 边距充足

### 音频
- [ ] 格式：MP3
- [ ] 采样率：22.05kHz
- [ ] 语速适中（1.0x）
- [ ] 音量平衡
- [ ] 有淡入淡出

### 视频
- [ ] 格式：MP4（H.264）
- [ ] 分辨率：1080p或720p
- [ ] 码率：2-3Mbps
- [ ] 文件大小≤50MB/分钟
- [ ] 设置封面
