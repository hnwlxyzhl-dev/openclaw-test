# 孤立森林 PPT v1 读者评审任务

## 你的角色
你是一个有基础数据分析背景、但不熟悉异常检测算法的研究生。这是这篇 PPT 的目标受众。

## 任务
请站在目标读者的视角，逐页阅读这份 PPT，评估你能否看懂、是否有收获。

## 必读文件
1. `/home/admin/.openclaw/workspace-weaver/output/isolation-forest_v1.pptx` — 用 python-pptx 读取所有文本内容
2. `/home/admin/.openclaw/workspace-weaver/memory/isolation-forest-ppt-content-logic.md` — 原始内容设计（对照参考）

## 评审维度（满分 100）

1. **内容深度**（20 分）— 是否过于简单？是否有足够的细节？
2. **逻辑连贯**（20 分）— 页面之间是否有自然的过渡？叙事是否流畅？
3. **通俗易懂**（20 分）— 你能否看懂？比喻是否贴切？还是反而让人困惑？
4. **信息密度**（15 分）— 每页是否有足够的信息量？是否有空洞页？
5. **整体收获**（15 分）— 读完 PPT 后是否有实质性收获？
6. **文字质量**（10 分）— 是否有重叠、错别字、不通顺的句子？

## 评审方法
1. 用 python-pptx 读取每一页的所有文本内容
2. 逐页评审，站在读者角度回答：
   - 这一页我能看懂吗？
   - 和上一页的衔接自然吗？
   - 比喻是否帮助理解？
   - 内容是否太浅？缺少什么关键信息？
   - 有没有排版问题影响阅读？
3. 给出总分和每页评审意见
4. 列出最需要改进的 3 个问题
5. 给出具体修改建议

## 评分标准
- 总分 ≥ 90：优秀，读者体验好
- 80-89：良好，有小问题需优化
- 70-79：一般，有明显短板需修复
- < 70：需要较大改动
