# 图片生成工具安装方案

## 当前系统环境

- Python版本：3.6.8
- 操作系统：Linux
- 如需安装Stable Diffusion，建议升级到Python 3.10+

## 推荐方案

### 方案A：在线工具（无需安装）⭐推荐

**优势：** 
- 无需配置环境
- 即开即用
- 不占用本地资源

**推荐工具：**
1. **Midjourney** - 通过Discord使用
2. **DALL-E 3** - 通过ChatGPT Plus
3. **Bing Image Creator** - 完全免费
4. **Leonardo.ai** - 每日150张免费
5. **DreamStudio** - Stable Diffusion在线版

**开始步骤：**
1. 访问 bing.com/images/create （免费）
2. 访问 leonardo.ai 注册账号（免费）
3. 如有需要，订阅Midjourney或ChatGPT Plus

### 方案B：本地安装（适合高频使用）

#### 1. 安装Python 3.10+

```bash
# Ubuntu/Debian
sudo apt update
sudo apt install python3.10 python3.10-venv python3-pip

# 创建虚拟环境
python3.10 -m venv sd-env
source sd-env/bin/activate
```

#### 2. 安装Stable Diffusion WebUI

```bash
# 安装依赖
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
pip install xformers

# 克隆项目
git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui.git
cd stable-diffusion-webui

# 运行
./webui.sh
```

#### 3. 下载模型

**推荐模型：**
- SDXL Base: https://huggingface.co/stabilityai/stable-diffusion-xl-base-1.0
- Realistic Vision: https://civitai.com/models/4201
- DreamShaper: https://civitai.com/models/4384

**存放位置：**
`stable-diffusion-webui/models/Stable-diffusion/`

### 方案C：ComfyUI（进阶用户）

```bash
# 克隆项目
git clone https://github.com/comfyanonymous/ComfyUI
cd ComfyUI

# 安装依赖
pip install -r requirements.txt

# 运行
python main.py --listen 0.0.0.0 --port 8188
```
