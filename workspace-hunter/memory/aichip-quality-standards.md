# A股AI芯片行业深度研究报告 PPT 质检标准

## 边界控制
- 幻灯片尺寸: 13.333 x 7.5 英寸
- 所有文本框: left+width ≤ 13.3, top+height ≤ 7.4
- 底部安全区: top ≥ 7.15 保留给页脚
- 右侧安全区: left+width ≤ 13.0

## 字体标准
- 正文: ≥ 9pt
- 表格内容: ≥ 9pt
- 标题: ≥ 12pt
- 封面标题: ≥ 28pt
- 目录编号: ≥ 10pt

## 内容标准
- 数据准确性（对照Eden提供的原始数据）
- 必须有具体数字
- 比喻适当（金融报告风格，不过度通俗）
- 表格数据完整

## 视觉标准
- 深色主题一致性（深蓝黑背景）
- 科技蓝/金色点缀配色统一
- 卡片边框整齐
- 页码显示正确（x/18格式）

## 历史教训（必须避免！）
- 文字超出画框
- 文本框高度异常
- 表格对齐问题
- 颜色不统一

## 检查方法
用 python-pptx 读取文件，逐页逐个文本框检查边界。
