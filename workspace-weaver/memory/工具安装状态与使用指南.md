## 💻 本地工具安装（个人电脑）

### 简单安装方案（Windows/Mac）

#### 1. Edge-TTS（推荐，免费）

**Windows安装：**
```powershell
# 1. 安装Python 3.10+（从python.org下载）

# 2. 安装edge-tts
pip install edge-tts

# 3. 测试
edge-tts --text "测试语音" --write-media test.mp3
```

**Mac安装：**
```bash
# 1. 安装Python（如未安装）
brew install python

# 2. 安装edge-tts
pip3 install edge-tts

# 3. 测试
edge-tts --text "测试语音" --write-media test.mp3
```

**Python使用：**
```python
import edge_tts
import asyncio

async def generate_speech():
    text = "欢迎观看本次演示"
    voice = "zh-CN-XiaoxiaoNeural"  # 晓晓

    communicate = edge_tts.Communicate(text, voice)
    await communicate.save("output.mp3")
    print("语音生成成功！")

asyncio.run(generate_speech())
```

## 🔧 故障排除

### Q1: Edge-TTS安装失败？
```
原因：Python版本过低
解决：安装Python 3.10或更高版本
下载：python.org/downloads
```

### Q2: Stable Diffusion很慢？
```
原因：没有GPU或显存不足
解决：
- 使用在线工具（Leonardo.ai）
- 或租用云GPU（RunPod、Vast.ai）
```

### Q3: 生成质量不满意？
```
解决：
- 优化Prompt描述
- 多次生成选择最佳
- 使用高质量模型
- 参考社区作品学习
```

### Q4: 在线工具访问慢？
```
解决：
- 使用网络加速工具
- 选择国内工具（Kling）
- 或使用本地工具
```
